# KnowYourself
> 익명의 속마음 전달 모바일 어플리케이션 

<br>

**개발 동기**
* 상대방이 생각하는 자신의 모습을 궁금해하는 사람은 많지만 본인이 생각하는 상대방의 모습을 거짓없이 이야기할 수 있는 사람은 많지 않다. <br> 본 프로젝트를 통해, 익명의 힘을 빌려 서로가 생각하는 상대방의 모습을 솔직하게 주고받음으로써 모두가 더 나은 모습으로 나아갈 수 있도록 도움을 주고자 하였다.
